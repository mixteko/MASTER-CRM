# Mini Farmacia

Objetivo:

Crear una app web sencilla para administrar una farmacia local.

Módulos:

- Dashboard
- Productos
- Inventario
- Ventas
- Clientes
- Pagos
- WhatsApp

Funciones principales:

- Agregar productos
- Editar productos
- Controlar existencias
- Registrar ventas
- Calcular total
- Registrar método de pago
- Preparar pedido
- Enviar resumen por WhatsApp

Tecnología inicial:

- HTML
- CSS
- JavaScript

Futuro:

- Supabase
- Usuarios
- Roles
- WhatsApp Cloud API
- Mercado Pago
- Clip
